# 项目运行指南

- 安装/更新依赖包：`npm install` 或 `yarn`
- 运行：
  - 启动为 dev 环境：`npm run dev` 或 `yarn run dev`
  - 打包为 prod 环境：`npm run build` 或 `yarn run build`

# git 

## type: commit 的类型

* feat: 新功能、新特性
* fix: 修改 bug
* perf: 更改代码，以提高性能
* refactor: 代码重构（重构，在不影响代码内部行为、功能下的代码修改）
* docs: 文档修改
* style: 代码格式修改, 注意不是 css 修改（例如分号修改）
* ci: 持续集成相关文件修改
* chore: 其他修改（不在上述类型中的修改）
